Quick Start Guide

Get up and running in minutes

1

Clone the repository

Start by cloning the project repository to your local machine.

git clone https://github.com/your-username/spring-boot-mybatis-seed.git cd spring-boot-mybatis-seed
2

Configure your database

Edit the application.properties file to set up your database connection.

spring.datasource.url=jdbc:mysql://localhost:3306/your_database spring.datasource.username=root spring.datasource.password=password spring.datasource.driver-class-name=com.mysql.jdbc.Driver
3

Build and run

Use Maven to build and run the application.

mvn clean compile mvn spring-boot:run
4

Generate code for your entities

Use the included code generator to create model, mapper, and service classes.

# Edit src/test/java/CodeGenerator.java with your table info # Then run the generator mvn test -Dtest=CodeGenerator
5

Enjoy rapid development

Your API is now up and running! Access it at http://localhost:8080
